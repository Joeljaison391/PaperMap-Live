## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run tauri dev

# Build for production
npm run tauri build
```

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Backend**: Tauri 2.0, Rust
- **Maps**: MapLibre GL JS with OpenStreetMap tiles
- **Rendering**: WebGL2 with hardware acceleration
